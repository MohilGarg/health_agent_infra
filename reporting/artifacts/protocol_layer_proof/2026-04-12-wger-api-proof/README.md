# wger API proof bundle

This bundle is mock-backed only.

It proves a bounded pull-and-transform surface against a disposable local mock of published wger endpoints, including:
- auth flow shape
- deterministic canonical ID replay
- idempotent second-sync behavior
- tombstone handling
- crash-resume behavior
- bounded canonical gym-domain outputs

It does **not** prove:
- a live self-hosted or Docker-backed wger runtime on this host
- flagship source-of-truth status for resistance training
- that the legacy trio day proof gate should be reopened as current flagship truth

For current doctrine, manual structured gym logs remain the source-of-truth path and this bundle should be read as exploratory non-flagship connector evidence only.
